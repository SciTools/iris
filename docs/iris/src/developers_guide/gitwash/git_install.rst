 Install git
=============

Overview
========

================ ===============================
Debian / Ubuntu  ``sudo apt-get install git``
Fedora           ``sudo yum install git``
Windows          Download and install msysGit_
OS X             Use the git-osx-installer_
================ ===============================

In detail
=========

See the git page for the most recent information.
